Untuk menambahkan gambar:

1. Simpan gambar yang diinginkan dengan nama "bye.jpg"
2. Pastikan gambar tersebut adalah file gambar yang valid (bukan folder)
3. Letakkan file tersebut di folder ini (C:\Piee\images\)
4. Refresh halaman web di browser

Catatan: 
- Pastikan nama file tepat "bye.jpg" (huruf kecil semua)
- File harus berupa gambar JPG yang valid
- Jangan buat folder dengan nama yang sama
